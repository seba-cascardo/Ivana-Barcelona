## Convención

Cada idea = una sección con:
- **Idea**: descripción clara.
- **Por qué**: problema que resuelve / oportunidad.
- **Costo estimado**: low / med / high.
- **Fase candidata**: Phase 2 / 3 / 4 / 5+.
- **Status**: backlog / under-consideration / accepted / rejected.

## Ejemplo de entrada (template)

### Música adaptativa por capas

- **Idea**: que la música de ciudad tenga capas que entran/salen según
  tensión narrativa (descubriste pista importante → entra string section).
- **Por qué**: aumenta inmersión, da feedback emocional sin texto.
- **Costo estimado**: high (requiere infra de audio buses + composer-friendly
  pipeline).
- **Fase candidata**: Phase 3.
- **Status**: backlog.
